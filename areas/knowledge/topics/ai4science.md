# AI4Science

## Definition

AI4Science is the use of AI systems to support scientific discovery, data processing, experiment automation, simulation, instrument operation, and research workflow design.

In this wiki, the topic is treated as a bridge between agent systems, tool use, data infrastructure, and automated experimentation.

## Why it matters

- It is a concrete domain where agent tool use must interact with real scientific workflows and data formats.
- It connects [[function-calling-and-tool-use|Function Calling and Tool Use]] with experiment automation.
- It provides a useful testbed for long-horizon agents, self-evolving systems, and data-side research infrastructure.
