## Chinese Summary / 中文摘要

### 保留原因

- 这篇保留了“connectome 约束建模 + 具身仿真 + 行为验证”这条真正有长期价值的研究主线，同时跳过了更像执行摘要的 companion note。
- 它对 AGI 的意义不在“复制果蝇”，而在于结构先验、模块化控制和具身验证方法。

### 核心结论

1. 当前最接近“脑上传”的，不是主观意识上传，而是保留连接结构、回路假设和身体耦合的可执行仿真。
2. 这条 pipeline 可以概括为：电子显微成像 -> connectome 重建 -> 全脑标注与细胞分类 -> 网络分析 -> 连接组约束模型 -> 全身物理仿真 -> 行为验证。
3. 果蝇是非常合适的实验对象：规模够大到有研究价值，同时还在当前重建能力范围内。
4. 对 AGI 真正有价值的是架构启发：稀疏拓扑、模块边界、瓶颈结构和生物约束可能本身就是强先验。
5. 具身不是可选附加项，脑和身体闭环之后才有机会真正验证控制策略。
6. 目前仍然缺少长期可塑性、神经调质、内部状态和记忆连续性，因此应视为可执行控制仿真，而不是完整心智上传。

## Retained takeaways

1. The closest current analogue to "brain upload" is not consciousness transfer.
   It is a structure-constrained executable simulation that preserves wiring, circuit dynamics assumptions, and body coupling well enough to reproduce behavior.

2. The working pipeline is:
   - electron microscopy
   - connectome reconstruction
   - whole-brain annotation and cell typing
   - network analysis
   - connectome-constrained neural model
   - whole-body physics simulation
   - behavioral validation

3. Fruit fly is a uniquely practical substrate for this line of work.
   The note highlights the adult whole brain scale at roughly `139,255` neurons and about `50 million` synapses, which is large enough to matter and still within current reconstruction range.

4. The AGI-relevant value is architectural, not mimetic.
   The durable lesson is that sparse topology, modular control loops, and biologically grounded bottlenecks may function as strong priors for executable intelligence.

5. Embodiment is part of the model, not an optional add-on.
   A connectome without a body can validate structure, but once coupled to a physics simulation it can begin to validate control policy and closed-loop behavior.

6. The current ceiling is still clear.
   Missing layers include long-timescale plasticity, neuromodulation, internal state, and memory continuity, so this should be treated as executable control simulation rather than full mind upload.
