## 这是什么
这不是一个传统软件项目仓库，而是一个持续演化的个人知识系统（personal wiki / second brain）。

它的目标是：
- 把分散的信息收拢成可检索、可回顾、可持续更新的结构
- 保留“当前状态”与“历史版本”两层信息
- 让训练计划、饮食方案、学习记录、工作笔记都能被长期复用
